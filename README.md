# H5 2D Game

A simple 2D survival game built with HTML5 Canvas and Vanilla JavaScript.

## Features

- **Player Movement**: Use `W`, `A`, `S`, `D` keys to move the player (blue circle).
- **Auto-Shooting**: The player automatically shoots bullets (yellow) at the nearest monster.
- **Monsters**: Monsters (red circles) spawn from the edges of the screen every 1.5 seconds (40 per minute).
- **Combat**: Monsters have 3 HP. Bullets deal 1 damage.

## How to Play

1. Open `index.html` in your web browser.
2. Survive as long as possible!

## Development

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  Open the URL shown in the terminal (usually `http://localhost:5173`).
